HTML TAGS

    Header Tags:
         h1,h2,h3,h4,h5,h6 are used to define HTML headings.
         h1 defines the most important heading.
         h6 defines the least important heading.
    Void element or Self closing element: 
         A self-closing tag in HTML is a kind of HTML tag that does not need to be closed 
         manually by its closing tag, which means it does not have a seperate closing tag.
         EX: <input/>, <hr/>, <br/>, <img/>
    <p> Tag: 
         The tag is used to represent a paragraph which means a group of sentences.
    <b> Tag:
         <b> Tag is used to display the written text in bold format.
         It does not have real semantic meaning. 
    <Strong> Tag: 
         The strong tag is used to indicate that the text within it is important or 
         should be emphasized. 
    <i> Tag:
         The italic tag is a span of text that represents a change in quality of text.
    <em> Tag:
         The emphasis tag is used to define emphasized text. The content inside is
         typically displayed in italic.
    <sup> Tag:
         The <sup> tag defines superscript text. Superscript text appears half a character 
         above the normal line, and is sometimes redered in a smaller font.
    <sub> Tag:
         The <sub> tag defines superscript text. Superscript text appears half a character 
         below the normal line, and is sometimes redered in a smaller font.
    <mark> Tag: 
         It is used to highlight some part of the text for quick reference inside another element
         such as in a paragraph that is of more importance or relevance to the user.
    <del> Tag: 
         The <del> tag is used to identify text that has been deleted from a document but 
         retained to show the history of modifications made to the document.
    <ins> Tag: 
         The <ins> tag defines a text that has been inserted into a document. 
         Browsers will usually underline inserted text.
    <u> Tag:
         The <u> tag represents some text that is unarticulated and styled differently from normal 
         text.  The content inside is typically displayed with an underline.
    <pre> Tag:
         The <pre> tag defines preformatted text. Text in a <pre> element is displayed in a fixed-width font, 
         and the text preserves both spaces and line breaks. The text will be displayed exactly as written in 
         the HTML source code.
    <small> Tag:
         <small> Used for specifying smaller text. 
    <abbr> Tag: 
         The <abbr> element is used along with a title attribute to associate a full-text explanation with an abbreviation 
         or acronym.
    Block level elements:  
         A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and 
         after the element. Two commonly used block elements are: <p> and <div> .
    inline elements:  
         Inline elements display in a line. They do not force the text after them to a new line. 
    BlockQuote:
         The <blockquote> tag specifies a section that is quoted from another source.
